# Caso de Uso: Autenticación y Acceso

**Objetivo:** Permitir a los [[Usuario]]s acceder de forma segura a la plataforma y gestionar aspectos básicos de su propia cuenta, como la contraseña.

**Actor Principal:** [[Usuario]]

**Actores Secundarios:** Ninguno específico, aunque la interacción puede involucrar sistemas de envío de notificaciones o servicios de autenticación.

**Descripción:** Este caso de uso describe el proceso mediante el cual un [[Usuario]] inicia sesión en la plataforma utilizando sus credenciales, así como los mecanismos disponibles para recuperar o cambiar su contraseña si es necesario. Permite a los usuarios finales gestionar su propio acceso al sistema.
